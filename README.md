# Study-Plus-Navigator

本项目是一个基于 JavaScript 的自动化技术研究工具，旨在探索 Web 页面 DOM 状态监测与交互逻辑的实现方案。

## 🔬 项目背景
本项目目前主要用于个人技术积累，专注于研究如何通过自动化脚本优化特定教育平台的交互流程。

## 🛠️ 环境要求
* **浏览器**：建议使用 Chrome 或 Edge (内核版本 100+)
* **内核插件**：Tampermonkey (或同类脚本管理工具)

## 📦 核心逻辑
1. **状态轮询**：每隔 1000ms 自动检测页面元素属性。
2. **事件驱动**：深度集成 `video` 标签的 `ended` 事件监听。
3. **自动化跳转**：当监测到学习进度达到 100% 或视频结束时，自动执行跳转逻辑。

## 🛠️ 安装指南 (纯小白版)

### 1. 安装“篡改猴”插件
首先，你的浏览器需要安装 **Tampermonkey** 插件。
* **Chrome/Edge 浏览器用户**：请前往应用商店搜索 `Tampermonkey` 并安装。
* 安装完成后，浏览器右上角会出现一个“黑色方块”图标。

### 2. 导入脚本
1. 点击浏览器右上角的 **Tampermonkey 图标**，选择“**管理面板**”。
2. 点击顶部的“**+**”号（或“添加新脚本”）。
3. 删掉编辑器里原有的所有代码。
4. 将本项目中的 `study-navigator.user.js` 代码全文**复制并粘贴**进去。
5. 按键盘上的 `Ctrl + S` 键保存（或者点击文件 -> 保存）。
