# Study-Plus-Navigator 课程辅助工具

本工具是一个基于 Tampermonkey（篡改猴）的浏览器脚本，旨在通过自动化技术辅助个人学习进度管理。

## 📖 核心功能
* **自动检测进度**：当课程学习进度达到 100% 时，自动触发跳转。
* **视频播放完毕跳转**：视频播放结束后，自动点击下一节。
* **手动开关**：页面右下角设有浮动按钮，可随时开启或关闭自动化功能。
* **智能避让**：在触发跳转的冷却期内，不会因页面抖动误触发。

## 🛠️ 安装指南（纯小白版）

### 1. 安装“篡改猴”插件
首先，你的浏览器需要安装 **Tampermonkey** 插件。
* **Chrome/Edge/浏览器用户**：请前往应用商店搜索 `Tampermonkey` 并安装。
* 安装完成后，浏览器右上角会出现一个“黑色方块”图标。

### 2. 导入脚本
1. 点击浏览器右上角的 **Tampermonkey 图标**，选择“**管理面板**”。
2. 点击顶部的“**+**”号（或“添加新脚本”）。
3. 删掉编辑器里原有的所有代码。
4. 将本项目中的 `study-navigator.user.js` 代码全文**复制并粘贴**进去。
5. 按键盘上的 `Ctrl + S` 键保存（或者点击文件 -> 保存）。

### 3. 开始使用
1. 打开你的学习平台课程播放页面。
2. 看到右下角出现红色“**⏭ 自动跳转**”按钮，即表示脚本已启动。
3. 脚本会自动监测进度，请保持页面窗口处于活动状态。
